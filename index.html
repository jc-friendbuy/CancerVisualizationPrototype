Cancer Visualization Prototype

This project aims to create tools for visualizing and analyzing cancer data from the Cancer Cell Line Encyclopaedia. This will hopefully help identify interesting patterns in cancer behavior, which will allow us to design more effective, personalized therapy strategies for dealing with each patient individually.

This project is a member of the University of Costa Rica's RIB group.

Please note: this is in (very) early alpha stage, and not intended for any sort of production use.
